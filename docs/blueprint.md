# **App Name**: Oke Raffle

## Core Features:

- Registration Form: Collect user information (email, full name) via a registration form.
- Unique Number Generation: Automatically assign a unique raffle number upon successful registration.
- QR Code Generation: Generate a QR code linking directly to the registration website.
- Confirmation Email: Send a confirmation email containing the user's raffle number.
- Admin Panel: A CMS allowing event staff to manage registrations and raffle settings.

## Style Guidelines:

- Primary color: Soft purple (#B19CD9) for a calm and inviting atmosphere.
- Background color: Light gray (#F0F0F0) for a clean, uncluttered look.
- Accent color: Deep blue (#3F51B5) to highlight key actions and information.
- Headline font: 'Poppins' (sans-serif) for a modern, geometric feel. Body font: 'PT Sans' (sans-serif) to ensure readability in longer text.
- Use simple, recognizable icons to represent different actions and information.
- A clear and intuitive layout with a focus on ease of navigation.
- Subtle animations and transitions to provide feedback and guide users through the registration process.
